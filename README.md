# personal_notes_microservice

The "personal_note_microservice" is the core module of the NotesSphere ecosystem.
It handles personal user data such as notes,tasks,reminders allowing individuals to manage their day-to-day productivity in a secure microservice.

--- 

#Features (Planned)
| Feature | Description | Status |
|----------|--------------|--------|
| ## Notes Management | Create, update, delete, and retrieve personal notes |  Model Added |
| ## Task Scheduler | Add tasks, deadlines, and reminders |  Model Added |
